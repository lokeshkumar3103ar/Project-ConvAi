# RTX Logs Directory Structure

This directory contains organized logs of changes made to the ConvAi-IntroEval application.

## Organization

Logs are organized in date-based subfolders with the naming convention `RTX_MMMDD_YYYY`.

Example: `RTX_June8_2025` contains all logs from June 8, 2025.

## Log Files

Each log file follows the naming pattern:
`RTX_[component]_[activity]_[monthname]_[day]_[time].md`

Examples:
- `RTX_concurrency_discussion_june_9_1430.md` - Discussion log
- `RTX_pipeline_implementation_june_9_1500.md` - Implementation log
- `RTX_testing_results_june_9_1630.md` - Testing results log
- `RTX_database_changes_june_9_1700.md` - Database modification log

Special files:
- `RTX_summary_all_changes_[monthname]_[day]_[time].md` - Daily summary
- `RTX_discussion_[topic]_[monthname]_[day]_[time].md` - Topic discussions
