What Happens Behind the Scenes

  1. Your file is uploaded to the server using a secure connection
  2. The server saves the file temporarily in the videos directory
  3. The Whisper AI model (size: turbo) processes your audio
  4. The transcription is segmented with proper timestamps
  5. The result is saved as a text file in the transcription directory
  6. The server returns both the file path and the formatted transcript
