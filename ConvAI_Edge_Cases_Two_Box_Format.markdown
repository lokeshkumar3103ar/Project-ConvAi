# ConvAI Edge Cases: 80 Test Cases in Two-Box Format

This document outlines 80 edge cases for the *ConvAI Form Filler* AI, which collects data for HR and healthcare forms via voice input. Each case is presented in a two-box format: the **left box** contains the AI’s question and the issue, and the **right box** contains the human’s response and the solution. The cases (40 HR, 40 Healthcare) cover linguistic, emotional, technical, ethical, and extreme scenarios, with hardcoded responses for the demo app, testable on an NVIDIA RTX 4060 with Mistral-7B and Whisper.

## HR Edge Cases

| **AI Question & Issue** | **Human Response & Solution** |
|-------------------------|-------------------------------|
| **Case 1: Irrelevant Answer**<br>**AI Question**: Tell me about yourself.<br>**Issue**: User provides off-topic info, delaying form completion. | **Human Response**: I love pizza and hiking on weekends.<br>**Solution**: Redirect with specific prompt: “Pizza’s great! What’s your name and a work skill?” Updates form with relevant data (e.g., Name: Arjun, Skills: Web development). Use spaCy to detect irrelevant topics. |
| **Case 2: One-Word Answer**<br>**AI Question**: What’s your main skill?<br>**Issue**: Minimal response lacks detail for form. | **Human Response**: Coding.<br>**Solution**: Ask closed-ended question: “What kind? Python or JavaScript?” Maps response to form (e.g., Skills: Python). Mistral-7B extracts single words. |
| **Case 3: Contradictory Answer**<br>**AI Question**: How old are you?<br>**Issue**: Conflicting info risks incorrect form data. | **Human Response**: I’m 25. Wait, no, 30.<br>**Solution**: Confirm latest input: “Are you 30?” Updates form (e.g., Age: 30). Log contradictions in MongoDB. |
| **Case 4: Angry Outburst**<br>**AI Question**: What’s your work experience?<br>**Issue**: Frustration may derail conversation. | **Human Response**: This is stupid, why do I need to do this?<br>**Solution**: Acknowledge and simplify: “I hear you. What’s one job you’ve had?” Updates form (e.g., Experience: Cashier, 1 year). Use spaCy for sentiment analysis. |
| **Case 5: Sad Response**<br>**AI Question**: Tell me about yourself.<br>**Issue**: Emotional state reduces cooperation. | **Human Response**: I’m not feeling great about my job search.<br>**Solution**: Empathize and pivot: “I’m sorry. | **Case 6: Overly Verbose**<br>**AI Question**: What’s your work experience?<br>**Issue**: Long stories complicate data extraction. | **Human Response**: I started as an intern, got promoted, worked on this huge project…<br>**Solution**: Request summary: “Can you sum up your main role?” Updates form (e.g., Experience: 5 years). Limit input to 500 tokens. |
